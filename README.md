# 📊 Student Grade Calculator

A simple **console-based Student Grade Calculator** developed in **Java** as part of my **CodSoft Java Programming Internship**. This application calculates a student's total marks, average percentage, and grade based on marks entered for multiple subjects.

## 🚀 Features

- 📚 Accepts marks for multiple subjects
- ➕ Calculates total marks
- 📈 Calculates average percentage
- 🏆 Assigns grades based on average percentage
- ✅ Validates user input (marks between 0 and 100)
- 💻 Easy-to-use console interface

## 🛠️ Technologies Used

- Java
- Scanner Class
- Loops
- Conditional Statements
- Input Validation

## 📂 Project Structure

```
Student-Grade-Calculator/
│── StudentGradeCalculator.java
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/Student-Grade-Calculator.git
```

2. Open the project folder.

3. Compile the program:

```bash
javac StudentGradeCalculator.java
```

4. Run the application:

```bash
java StudentGradeCalculator
```

## 🎯 Grade Criteria

| Average Percentage | Grade |
|--------------------|-------|
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| 40 - 59 | E |
| Below 40 | F |
